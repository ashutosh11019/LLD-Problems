# Low Level Design (LLD) Problems Repository

Welcome! This repository is a collection of Low Level Design (LLD) solutions for various system design problems, implemented in Java. Each problem is structured with its own codebase, design documentation, and diagrams.

## 📂 Repository Structure

The repository is organized by problem statements. Each folder contains:
-   **Discussion.md**: Detailed problem statement, requirements, and LLD analysis.
-   **Source Code**: Java implementation of the design.
-   **Class Diagram**: Mermaid diagrams visualizing the architecture.

## 🛠️ How to Run

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Navigate to a problem directory**:
    ```bash
    cd <Problem_Folder_Name>
    # Example: cd Parking_Lot
    ```
3.  **Compile and Run**:
    Most solutions include a `Main.java` entry point.
    ```bash
    javac Main.java
    java Main
    ```

## 📝 Contribution

Feel free to contribute by adding new LLD problems or optimizing existing solutions!
1.  Create a new folder for the problem.
2.  Add a `Discussion.md` with the problem statement and design.
3.  Implement the solution.
4.  Update this `README.md` with the new problem link.
